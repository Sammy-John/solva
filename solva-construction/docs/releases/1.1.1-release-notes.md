# Release 1.1.1 Notes

**Version:** `1.1.1`
**Date:** `2026-04-16`

## Summary
This patch release focuses on schedule editing usability and dependency reliability. It improves task creation and move guidance in the planning table, restores correct backward dependency cascading, and adds clearer in-context feedback when a task date cannot move earlier because of dependency constraints.

## Included Changes
- Added `Task` action column before `Move`.
- Reduced `Status` column width for a little more table space.
- Added inline `+` task action for creating a new task directly below the selected row in the same section.
- Improved Move mode clarity with a more visible centered popup while keeping the schedule interactive.
- Fixed auto-shift dependency cascades so downstream tasks can move earlier when upstream tasks move earlier or shorten.
- Added blocked-edit callout for constrained start-date edits with:
  - earliest allowed start date
  - blocker task name
  - direct jump-to-blocker action
